In learning about Data types
Everything in python is an object and the data types too are objects of classes. 

NUMERIC DATA TYPES
These include Integers, float and complex.
Integers are whole numbers, floating point numbers contain decimals and complex numbers are of the form (x+yj) where x is the real part and yj is the imaginary part.

DICTIONARIES:
These include key-value pairs, that is to say the keys must pair with the values to be a dictionary. 
values can be repeated but keys cannot. Python dictionaryies are case sensitive, KEY is not the same as key.

ARRAYS
List
A list is a collection of items that is ordered and mutable.
A list is created using square brackets.
Tuple
A tuple is a collection of items that is ordered but not mutable. Once they are created, items in a tuple cannot be changed, added to or removed. A tuple is created in round brackets.
String
A string is a sequence of characters, it is used to represent text. Python does not have a character type, it only has a string of length 1. Strings can be created using one, two or three quotation marks.

SETS
A set is an unordered collection of unique immutable items. When working with sets, repeated items are automatically removed from the collection. 

BOOLEAN 
A boolean datatype stores True or False values. 

TYPE CONVERSION IN PYTHON.
Type conversion is of two types in python, the implicit and the explicit. The implicit type conversion is done during run time, the explicit is done by the programmer where he changes the type of an object in python so that it is suitable for the operations he intends to do.
